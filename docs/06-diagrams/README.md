# Diagrams

System diagrams, workflows, sequence diagrams, and visual documentation. Store all visual documentation assets here. Use Mermaid for text-based diagrams where possible.

## Guidelines

- Prefer Mermaid (`.md` with fenced code blocks) for diagrams that benefit from version control diffs
- Use descriptive filenames that reflect the diagram's subject (e.g., `auth-flow-sequence.md`)
- Keep exported image assets (PNG, SVG) alongside their source files when applicable
